Aula de Especificidade CSS

Este título é um h1, também tem a classe .titulo e o ID #titulo-principal.

O CSS tentou aplicar três cores nele: h1 queria azul, .titulo queria verde, e #titulo-principal queria roxo.

Resultado: ficou roxo, porque o ID tem mais especificidade.

1. Classe + Hover

Este card usa .card-hover no estado normal.

Ao passar o mouse, entra .card-hover:hover.

O hover vence porque é uma situação mais específica.

2. Duas Classes

Este card tem duas classes: caixa e destaque.

A regra .caixa.destaque é mais específica que apenas .caixa.

Passe o mouse para ver .caixa.destaque:hover.

3. ::before

O texto azul acima foi criado pelo CSS com ::before.

Ele não está escrito no HTML.

Passe o mouse e veja o conteúdo mudar.

4. ::after

Este card usa ::after para criar uma barra embaixo.

Passe o mouse para a barra aparecer.

Parece mágica, mas é só CSS se achando artista.

5. ID vence Classe

Passe o mouse aqui

A cor final normal é roxa, porque o ID vence. No hover, o ID com :hover deixa vermelho.
